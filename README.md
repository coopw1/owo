## Running from source

1. Download and install [AutoHotKey v2](https://www.autohotkey.com/download/)
2. Clone/[Download](https://github.com/coopw1/owo/archive/refs/heads/master.zip) this git repo
3. Run owo.ahk with AutoHotKey!
